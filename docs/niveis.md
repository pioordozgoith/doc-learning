# Nivel 01

## Headings

- **'#'** represents ```<h1>```
- **'##'** represents ```<h2>```
- **'###'** represents ```<h3>```
- **'####'** represents ```<h4>```
- **'#####'** represents ```<h5>```
- **'######'** represents ```<h6>```

## Paragraph

É so escrever alguima coisa que ele ja entende como ```<p>```

## Line

é so digitar  ```---```, isso vai criar uma linha na leta

---

## Quote

quando for parafrasear algo use ```>``` no inicio da frase, como se estivesse falando algo
